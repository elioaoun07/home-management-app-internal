## 1 · The five forms

`Feature Doc.md` (the vault-doc scaffold; referenced by CLAUDE.md's documentation rules) · `Pattern.md` (the 09-shelf form — underused because the shelf is underfed, [09 FABLED 2](<../../09 - Patterns & Lessons/FABLED 2/_index.md>)) · `Session - Feature/Bug Fix/Refactor.md` (the 08-Sessions forms — **whose target directory doesn't exist**; see below).

## 2 · How documents actually get created in 2026

By **copying the best live example**: campaign folders copy each other's uniform layout; FABLED 2 copied FABLED v1's shape; Atlas entries copy `_Template.md` (the one template that *does* run at volume — because a script and a hard rule enforce it). The lesson the Atlas teaches: templates get used when something *points* at them at creation time (a script, a rule, a checklist step) — not because they exist.

## 3 · One verified oddity

CLAUDE.md's documentation table routes session notes to `ERA Notes/08 - Sessions/{Features|Bug Fixes|Refactors}/` — **that directory does not exist** in the vault (verified 2026-07-02; the tree jumps 07 → 09). The three Session templates are forms for a filing cabinet that was never installed. In practice, session history lives in campaign Implemented Notes (better, since Hard Rule #25 enforces them) — the session-note *genre* was quietly superseded.
