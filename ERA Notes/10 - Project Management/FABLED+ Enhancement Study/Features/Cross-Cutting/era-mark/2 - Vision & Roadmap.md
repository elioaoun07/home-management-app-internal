# ERA Mark · Vision & Roadmap

> [FABLED+ root](<../../../_index.md>) · **ERA Mark** · [1 · Feature State](<1 - Feature State.md>) · [2 · Vision & Roadmap](<2 - Vision & Roadmap.md>) · [3 · Action Plan](<3 - Action Plan.md>) · [4 · Checklist](<4 - Checklist.md>)

## Enhancement thesis

Turn the mark into calm system language: a small finite state grammar with redundant text/haptic cues, strict motion budget, and no false certainty.

## Business and household value

Trustworthy presence makes ERA feel coherent without increasing notification volume. Accessibility and motion discipline prevent brand delight from becoming friction.

Measure attention returned, risk reduced, accessibility, and outcomes—not engagement.

## Roadmap

1. Now — inventory every mark state and user interpretation.
2. Next — standardize a finite semantic state set with reduced-motion alternatives.
3. Later — retain only ambient signals proven useful and calm.

## New opportunity set

### V1 — Semantic state grammar

- **Mechanism:** Define idle, listening, processing, needs-input, uncertain, offline, success, and failure with exact entry/exit rules.
- **Smallest proof:** Map current assistant surfaces to the grammar.
- **Success measure:** Users identify states correctly without guessing.
- **Kill criterion:** Reduce to fewer states if distinctions blur.
- **Invariant:** Visual state never claims success before domain confirmation.

### V2 — Calm ambient pulse

- **Mechanism:** Represent only one low-frequency household condition such as pending attention, with opt-in and no sensitive detail.
- **Smallest proof:** Trial a single pending-sync/attention state.
- **Success measure:** It informs without prompting compulsive checking.
- **Kill criterion:** Remove if distracting or redundant.
- **Invariant:** No privacy-sensitive state on public/locked surfaces.

### V3 — Motion energy budget

- **Mechanism:** Cap duration, frequency, GPU cost, and simultaneous animations per surface with reduced-motion equivalence.
- **Smallest proof:** Profile Hub and watch variants.
- **Success measure:** No dropped interaction/battery regression and full semantic parity without motion.
- **Kill criterion:** Use static states where animation adds no comprehension.
- **Invariant:** Accessibility never depends on motion or color alone.

## Existing-roadmap boundary

Voice personas and broad theme contrast work are prior ideas; this pack makes the mark semantically trustworthy.

## Strategy guardrail
