# ERA Mark · Action Plan

> [FABLED+ root](<../../../_index.md>) · **ERA Mark** · [1 · Feature State](<1 - Feature State.md>) · [2 · Vision & Roadmap](<2 - Vision & Roadmap.md>) · [3 · Action Plan](<3 - Action Plan.md>) · [4 · Checklist](<4 - Checklist.md>)

## Objective

Prove one cross-cutting improvement without destabilizing every consumer.

## Now

- Inventory mark usages, animations, and actual state meanings.
- Define semantic grammar and forbidden false states.
- Add reduced-motion/text equivalence requirements.

**Gate N:** deterministic inventory or read-only proof.

## Next

- Migrate one assistant surface and one status surface.
- Run comprehension and performance checks.
- Remove decorative state collisions.

**Gate X:** vertical slice across representative mobile/desktop, both viewers/themes, and degraded states.

## Later

- Trial one calm ambient signal.
- Measure distraction and comprehension.
- Keep the state set intentionally small.

**Gate L:** measured outcome earns wider adoption; otherwise stop at the proven slice.

## Mandatory preflight

- Trace consumers and route classes.
- Run Design Doctrine Ten Questions.
- Test accessibility, identity, cache, offline, and privacy behavior.
- Ratchet rather than bulk-rewrite.

## Non-goals

Voice personas and broad theme contrast work are prior ideas; this pack makes the mark semantically trustworthy.
