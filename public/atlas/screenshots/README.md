# Atlas screenshots

Place images here for the in-app `/atlas` viewer.

Naming convention:

- `<slug>-mobile.png` — mobile viewport (≤ 480 wide preferred)
- `<slug>-desktop.png` — desktop viewport
- `<slug>-detail-<n>.png` — extra shots (modals, drawers, alternate states)

`<slug>` must match the `slug` frontmatter of the corresponding entry in
`ERA Notes/04 - UI & Design/Page & Feature Atlas/`.

The Atlas UI shows a placeholder when an expected file is missing; just drop new files
in here over time — `pnpm atlas` will pick them up automatically.
